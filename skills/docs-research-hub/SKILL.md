# Docs Research Hub

Use this as the first docs or research skill call. Load only the focused skill(s) needed for the task:

- General research evidence, documentation search, repository scans, and codebase tours: `research-core`
- Scientific database queries (PubMed, USPTO), gget package workflows, literature reviews, and academic scholar evaluations: `scientific-research`
- Long-form articles, brand voice development, marketing campaigns, content engines, SEO, and social platform publishing: `content-core`
- Video rendering (Remotion, Manim), video database querying, media generation (Fal AI), and document parsing: `media-doc-processing`

Keep separate:
- Market, competitive, investor, and prediction-market research: `market-research`, `investor-materials`, `investor-outreach`, `prediction-market-oracle-research`, `ito-market-intelligence`
- Code and project explanation architectural records: `architecture-decision-records`

Prefer primary sources when accuracy matters, keep citations close to claims, and separate source facts from synthesis.
