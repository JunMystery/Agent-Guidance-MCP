# Frontend Hub

Use this as the first frontend skill call. Load only the focused skill(s) needed for the task:

- UI/UX direction, branding, visual systems, slides, typography, color, dashboards, or landing pages: `ui-ux-pro-max`
- Frontend architecture, core component patterns, dashboard building, and design auditing: `frontend-core`
- Framework-specific implementation (React, Next.js, Vite, Nuxt, Angular, Vue, SwiftUI, Flutter): `frontend-frameworks`
- Accessibility, WCAG, ARIA, keyboard, and screen-reader behavior: `frontend-a11y`, `accessibility`
- Browser verification and screenshots: `browser-qa`
- Motion design, transitions, micro-animations, and dynamic UI styling: `motion-design`
- Presentations, slides, and video rendering: `presentation-ui`

Keep the first pass focused on user workflow, visual hierarchy, accessibility, responsive behavior, and verification in the running app.
