# Skill Grouping Audit

[Back to README](../README.md)

This audit records the grouping-first strategy for the local `skills/` catalog. Hubs route; specialized skills teach. No specialized skill is deleted, renamed, or merged in this pass.

Future merges must follow the [Skill Merge Policy](skill-merge-policy.md).
