# v1.0.4 — 2026-07-21

## Added
- **Workflow Hardening & Feedback Loops:** Introduced programmatic lifecycle state tracking: `Context -> Plan -> Ask/Revise -> Build -> Test/Recheck -> Fix -> Document/Proposal`.
- **Planning Loop Validation:** Auto-detects human approval keywords (`ok`, `proceed`, `bắt đầu`...) to transition from `Ask_Revise` to `Build` stage. Soft-blocks code modifications before approval.
- **Circuit Breaker:** Limit to 3 consecutive fix attempts. If exceeded, automatically falls back to `Ask_Revise` stage to ask user for strategy revision.
- **Soft Gates Warnings:** Gated tools dynamically report workflow stage mismatches in stderr.
- **Integration Evals:** Added unit testing for workflow lifecycle controls, loop validation, and circuit breakers.
