# v1.0.2 — 2026-07-20

### Changed
- Version bump to 1.0.2 for `uv tool update` change detection

### Fixed
- Embedding daemon timeouts: increased HTTP timeouts from 3s to 30s for `/embed`, `/register`, `/api/backend`, `/unregister`
- Embedding daemon crash on Windows: `uvicorn.run(fd=...)` replaced with platform-specific `host:port` binding to avoid `socket.AF_UNIX` error
- LLM selector blocking search: wrapped `select()` with 2s thread timeout so cold model load doesn't block guidance search
- LLM selector cold start: added background pre-warm in `create_server()` so model loads before first search call
- Daemon first-request latency: pre-warm embedding model on startup via `@app.on_event("startup")` background thread
- LLM agent tool naming: added explicit instructions to use exact tool names from the available-tools list (fixes prefix-dropping errors on OpenCode)
- Tool prefix note in AGENTS.md: documented that some hosts prefix tool names with `agent-guidance-mcp_`
- `install.ps1` UTF-8 BOM: removed BOM so `irm ... | iex` works without parsing error
- Changed `AGENT_INSTRUCTIONS` in `server.py` to warn about tool name prefixing
- Changed "per cell" to "per call" typo in AGENTS.md
