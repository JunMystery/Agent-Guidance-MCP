# v1.0.5 — 2026-07-21

## Added
- **Workflow Gate Tool:** Added a lightweight `workflow_gate` MCP tool (`action="status"|"check"|"set_stage"`) to enable step-by-step stage verification and parsing user message approvals (`proceed`, `ok`, `bắt đầu`...).
- **Behavioral Enforcer Prompts:** Redesigned `.cursorrules`, `AGENTS.md` and rules generator in `deploy_rules.py` to command AI agents to call `workflow_gate` at the beginning of each turn before other tool calls.
- **Workflow State Management:** Gated checks in `server.py` now warn and block file access if the agent transitions to `Build` stage without approval or fails to report approvals.
- **Testing:** Adapted and verified tests covering the workflow gate loops and circuit breaker logic.
