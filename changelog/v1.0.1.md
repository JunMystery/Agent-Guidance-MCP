# v1.0.1 — 2026-07-20

### Added
- Pre-warm embedding model on daemon startup to eliminate first-request latency
- Pre-warm LLM skill selector in background thread at server start
- Windows compatibility for uvicorn embedding daemon (avoid `AF_UNIX` crash)
- 30-second timeout for daemon HTTP requests (was 3s, fixing false timeouts on cold start)

### Changed
- Merged `workflow`/`workflow_prompt`/`precode_check`/`verify`/`feedback` tools into `guidance(operation=...)` — tool surface reduced from 13 to 9
- LLM selector select() wrapped with 2s thread timeout to avoid blocking search
- Dashboard `/api/dirs` path-scoped to project root (security hardening)
- Bumped version from 1.0.0 to 1.0.1 for `uv tool update` change detection

### Fixed
- `recommend` guidance operation returning `None` (dead fallback code)
- `search(None)` / `reason(None)` crashing (missing guard)
- Server `AGENT_INSTRUCTIONS` listing stale tool names
- Dashboard `index.html` listing removed `workflow_prompt` tool
- Stale `workflow_prompt` reference in README priority enforcement table
- AGENT_INSTRUCTIONS docstring missing 4 new guidance operations
